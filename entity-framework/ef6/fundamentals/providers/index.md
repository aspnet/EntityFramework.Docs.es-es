### <a name="config-file-registration"></a><span data-ttu-id="becf1-136">Registro en el archivo de configuración</span><span class="sxs-lookup"><span data-stu-id="becf1-136">Config file registration</span></span>

``` xml

       <providers>

       </providers>
    </entityFramework>
```

<span data-ttu-id="becf1-138">Tenga en cuenta que, a menudo, si el proveedor de EF se instala desde NuGet, el paquete NuGet agrega automáticamente este registro al archivo de configuración.</span><span class="sxs-lookup"><span data-stu-id="becf1-138">Note that often if the EF provider is installed from NuGet, then the NuGet package will automatically add this registration to the config file.</span></span> <span data-ttu-id="becf1-139">Si instala el paquete NuGet en un proyecto que no es el proyecto de inicio de la aplicación, es posible que tenga que copiar el registro en el archivo de configuración del proyecto de inicio.</span><span class="sxs-lookup"><span data-stu-id="becf1-139">If you install the NuGet package into a project that is not the startup project for your app, then you may need to copy the registration into the config file for your startup project.</span></span>
