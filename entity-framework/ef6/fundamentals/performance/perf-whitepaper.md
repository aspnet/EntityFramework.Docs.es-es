#### <a name="322-cache-eviction-algorithm"></a><span data-ttu-id="9f720-388">3.2.2 algoritmo de expulsión de caché</span><span class="sxs-lookup"><span data-stu-id="9f720-388">3.2.2      Cache eviction algorithm</span></span>
